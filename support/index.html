Kosmorro Support

Getting started with Kosmorro or Kosmorrolib, but lost?

There are multiple ways to get help!

About version support

Because Kosmorro and Kosmorrolib are two specific projects that don't have the same goal, they don't benefit of the same version support. See this page for more information.

Getting support

On GitHub

If you have an account on GitHub, feel free to ask your questions in the Discussions tab of the project.
