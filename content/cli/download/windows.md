# Download Kosmorro on Windows

Kosmorro can run through the Windows Subsystem for Linux (WSL).  
Yup, that's cheating.

1. Download Debian from the Windows Store.
2. Start Debian from the Start menu, and follow the configuration instructions in the terminal.
3. Follow the instructions to [install Kosmorro on Debian](@/cli/download/linux.md#debian-ubuntu-linux-mint).

Note: by default, Debian is in English. If you want to configure another language, see [this tutorial](https://www.shellhacks.com/linux-define-locale-language-settings/).
