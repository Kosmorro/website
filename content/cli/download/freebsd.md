# Download Kosmorro on FreeBSD

Kosmorro is available in [your official repositories](https://www.freshports.org/astro/kosmorro)! [![Badge reporting the version of Kosmorro on FreeBSD's repositories](https://repology.org/badge/version-for-repo/freebsd/kosmorro.svg)](https://repology.org/project/kosmorro/versions)

You can install it directly with your package manager:

```bash
pkg install kosmorro
```
